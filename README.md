## 仿牛客社区
### 线上地址
- http://106.55.174.13/

### 项目简介
该项目页面结构主要来自牛客网社区，技术上采用基于SSM框架技术，以MySQL的作为数据库存储方案，并使用Redis、Kafka、ElasticSearch等中间件，旨在实现一个高性能高可用的开发者社区。

### 涉及主要技术
- SSM：项目基本架构，使用SpringBoot
- Spring Security：页面访问权限管理
- MySQL：数据库主要存储
- Redis：热榜分布式缓存、分布式session、登录ticket、站点统计
- Kafka：关注、喜欢、评论等系统通知
- Elasticsearch：全文搜索功能
- Quartz：热榜本地缓存定时过期、热榜分布式缓存定时更新、帖子分数定时更新
- Caffeine：本地缓存
- 云服务器

### 模块设计
1. 用户模块
    1. 用户注册(邮箱)
    2. 用户登录、退出
    3. 账号设置(头像、密码管理)
    4. 个人主页(个人信息、我的帖子、我的回复)
2. 帖子模块
    1. 帖子浏览
    2. 帖子发布
    3. 帖子评论
    4. 敏感词过滤
3. 互动模块
    1. 私信功能
    2. 互关功能
    3. 互赞功能
    4. 系统消息
4. 权限管理
    1. 用户权限(普通用户、版主、管理员)
    2. 操作权限(置顶、加精、删除)
    3. 访问权限(未登录、普通用户、版主、管理员)
5. 其他模块
    1. 热榜排行
    2. 帖子分享
    3. 全文检索
    4. 网站数据统计
    5. ...
